## ⚡ THE MISSION
Pomodoro Command Unit designed for developers who demand a tactical edge in their workflow. This isn't just a timer—it's a mission control center for your focus.
---
## 🛠️ SYSTEM SPECIFICATIONS
* Adaptive HUD: Fully responsive CSS Grid/Flexbox architecture that scales from 4K tiling setups to mobile displays using vmin fluid typography.
* Focus Lock (Visibility API): The system automatically halts execution if you navigate away from the tab, flashing DON'T LOSE FOCUS to maintain discipline.
* Objective Matrix: Replaces passive logs with a dedicated mission parameter input. Define your goal, initiate the session, and the system locks the input until the mission is complete.
* Zero-Latency Logic: Powered by useReducer and useRef to eliminate React render loops and ensure millisecond-perfect timing.
* Aesthetic: JetBrains Mono typography, CRT scanlines, hexagonal grid overlays, and holographic pulse animations.
---
## 🚀 DEPLOYMENT
### 1. Clone the Repository
```bash
git clone https://github.com/bhavi-th/pomodoro.git
cd pomodoro/
```
### 2. Install Dependencies
```bash
npm install
```
### 3. Launch Core
```bash
npm run dev
```

## ⌨️ HOTKEYS
* [ SPACE ]: INITIATE / HALT SESSION
* [ R ]: SYSTEM REBOOT (Hard Reset)

## 📡 PROTOCOLS
* 01_DEEP_WORK: 25-minute high-intensity focus.
* 02_REGEN_PHASE: 5-minute cognitive recovery.
* 03_COOLDOWN: 15-minute extended system rest.
